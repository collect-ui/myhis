# 03 新增菜单与页面标准步骤

标准落地顺序：

1. 新增模型并注册到 `model/base/add_table.go`
2. 新增后端 service 目录和 SQL
3. 把 service 挂到对应模块的 `service.yml`
4. 新增页面 JSON
5. 在 `collect/frontend/page_data/index.yml` 注册 `frontend.*`
6. 准备建表 SQL 和菜单 seed SQL
7. 编写接口和页面验收脚本
8. 更新需求文档和标准文档

菜单 seed 的最低字段：

- `menu_name`
- `menu_code`
- `url`
- `api`
- `router_group`
- `parent_id`
- `belong_project`

如果菜单来自数据库，不直接改仓库里的数据库文件；改为提供可重复执行的 seed 脚本。
