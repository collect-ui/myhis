# 05 联调与测试标准

统一目录：

- 脚本：`test/lowcode-page/scripts/`
- 最新结果：`test/lowcode-page/results/latest/`
- 历史归档：`test/lowcode-page/results/history/<feature>/<timestamp>/`

本需求对应脚本：

- `apply_mail_account_seed.sh`
- `backend/mail_account_api_check.sh`
- `frontend/mail_account_page_check.js`
- `run_mail_account_check.sh`

接口测试格式统一为：

- URL 使用 `?service=...`
- body 中保留 `service`
- 业务字段与 `service` 同级

页面验收默认地址：

- 页面基地址：`http://192.168.232.130:8015/`
- 目标路由：`http://192.168.232.130:8015/collect-ui#/collect-ui/framework/mail_account`

说明：

- 根地址 `/#/framework/mail_account` 在当前前端运行时会被默认首页重定向接管
- 无头浏览器验收应直接访问 `collect-ui#/collect-ui/...` 形式的真实路由

结果产物建议包含：

- `summary.md`
- `backend.log`
- `frontend.log`
- `api-response.json`
- `page-after-import.png`
- `console-errors.log`
- `checklist.md`
