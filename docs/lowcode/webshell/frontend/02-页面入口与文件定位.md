# 页面入口与文件定位

## 1. 主配置入口

- `collect/frontend/page_data/data/server/webshell.json`

这是 webshell 前端低代码唯一主配置，建议任何改动先在此文件按关键词定位。

## 2. 快速定位关键词

- `workspaceVisible`：工作空间 drawer 开关
- `workspaceDrawerWidth`：drawer 宽度
- `workspaceOptions`：项目列表
- `workspaceLoadedMap`：已加载项目
- `tag: "tree"`：目录树
- `tag: "editor"`：编辑器
- `rightMenuAction`：分割逻辑
- `removeAction`：关闭标签逻辑

## 3. 前端渲染相关源码（collect-ui）

- `F:/collect-ui/src/components/tabs/tabs.tsx`
- `F:/collect-ui/src/components/tree/tree.tsx`
- `F:/collect-ui/src/components/render/render-child.tsx`
- `F:/collect-ui/src/action/arr-push.tsx`

## 4. 常见改动落点

1. 改项目切换：drawer 标题区域 `listview.rowClickAction`
2. 改工作空间内容：drawer `tabs.itemAttr.children[0].layout-fit`
3. 改文件打开：tree `selectAction`
4. 改分割行为：editor tabs `rightMenuAction`
5. 改关闭行为：editor tabs `removeAction`
