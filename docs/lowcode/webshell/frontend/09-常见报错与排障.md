# 常见报错与排障

## 1) `newProps.treeData is not iterable`

- 原因：`treeData` 不是数组
- 修复：保证 `workspaceTree` 永远是数组

## 2) `Cannot read properties of undefined (reading 'path')`

- 原因：action 上下文没有 `row`
- 修复：确认当前 action 是否由 list/tabs 行上下文触发；必要时加空值保护

## 3) 关闭一个 tab，另一个面板内容消失

- 原因：同路径 editor 实例复用或关闭清理策略不当
- 修复：editor path 加 token；关闭仅清理空分组

## 4) `warn is not a function`

- 原因：调用 `message.warn`，环境只提供 `message.warning`
- 修复：统一改 `warning` 或兼容映射
